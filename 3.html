java script에 대하여

An object-based scripting programming language designed for dynamic handling of specific events or input values ​​from a user on a web page. It is mainly used as a way to improve the user experience. It is usually embedded in HTML documents and runs in the browser.




For example, when you move the mouse over a specific image or text in an HTML document, the image size increases or the text changes color. In addition, if you type "za" into the search box input box, you can also use JavaScript to display a list of search terms that start with "za", which is currently searching for many users.
